## MongoDB分片集群特点
- 应用全透明，无特殊处理
- 数据自动均衡
- 动态扩容，无须下线
- 提供三种分片方式

## 分片集群数据分布方式
- 基于范围
- 基于Hash
- 基于zone / tag
